pub struct SgxFile { /* private fields */ }
Expand description

A reference to an open Sgxfile on the filesystem.

An instance of a SgxFile can be read and/or written depending on what options it was opened with. SgxFiles also implement [Seek] to alter the logical cursor that the file contains internally.

SgxFiles are automatically closed when they go out of scope.
