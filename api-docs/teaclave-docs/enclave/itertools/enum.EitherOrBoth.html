pub enum EitherOrBoth<A, B> {
    Both(A, B),
    Left(A),
    Right(B),
}
Expand description

Value that either holds a single A or B, or both.

Variants

Both(A, B)

Both values are present.

Left(A)

Only the left value of type A is present.

Right(B)

Only the right value of type B is present.
